# 💼 Job Portal Web Application (MERN Stack)

A full-stack **Job Portal Web Application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**. This platform allows users to search and apply for jobs, while recruiters can post and manage job listings.

---

## 🚀 Features

### 👨‍💻 For Job Seekers

* 🔍 Search jobs with filters
* 📄 Apply for jobs
* 📂 Upload resume
* 👤 Manage profile
* 📌 Track applied jobs

### 🧑‍💼 For Recruiters / Admin

* ➕ Post new job listings
* ✏️ Update & delete jobs
* 👀 View applicants
* 📊 Manage job postings

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* React Router

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication
* Multer (for file uploads)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/amanrawat08/Job-portal.git
cd Job-portal
```

### 2️⃣ Install Dependencies

#### Backend

```bash
cd server
npm install
```

#### Frontend

```bash
cd client
npm install
```

---

### 3️⃣ Environment Variables

Create a `.env` file in the server folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLOUDINARY_NAME=your_cloud_name
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
```

---

### 4️⃣ Run the Application

#### Backend

```bash
cd server
npm run dev
```

#### Frontend

```bash
cd client
npm run dev
```
